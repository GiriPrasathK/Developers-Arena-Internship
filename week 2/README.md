# Student Grade Calculator

## Week 2 Project – Control Flow & Data Structures

This project is a Python-based Student Grade Calculator developed as part of Week 2 practical work.  
It demonstrates the use of control flow statements, functions, lists, dictionaries, and basic input validation.

---

## Features
- Accepts multiple students' details
- Validates marks input (0–100)
- Calculates average marks
- Assigns grades based on average
- Displays individual results and class statistics

---

## Grade Criteria
| Average Marks | Grade |
|--------------|-------|
| ≥ 90         | A     |
| 80–89        | B     |
| 70–79        | C     |
| 60–69        | D     |
| < 60         | F     |
