# Library Management System (OOP)

## Description
A Python-based Library Management System built using Object-Oriented Programming principles.
It supports book borrowing, member management, due date tracking, file persistence, and backups.

## Features
- Book, Member, Library classes
- Borrow/Return with overdue calculation
- JSON data storage
- Backup system
- Menu-driven interface
- Unit tests using pytest

## How to Run
```bash
python library_system/main.py
